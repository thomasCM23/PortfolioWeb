Text Generation

Following This Tutorial

In the aforementioned tutorial recurrent neural networks are used for text generation, using a vocabulary containing the english alphabet, numeric characters 0-9, and common punctuation. The tutorial follows Andrej Karpathy's The Unreasonable Effectiveness of Recurrent Neural Networks, which goes into details on how to use RNNs for text generation and how effective they are at sequence processing tasks.


Using tensorflow, keras, and the tutorial, I was able to create checkpoints, which I later use for inference via an API which you can try below. I trained the network on the same data set as mentioned in the tutorial, but modified the network architecture and the amount of time trained.
